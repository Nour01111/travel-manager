#projet travel_manager

  Travel-Manager est une simple application cmd pour la gestion d’une agence de voyage écrit en
  langage C. Il permet de voir tous les voyages disponibles entre deux dates et gère les dates des
  voyages.

##installation
  il suffit d avoir les fichiers dans un meme dossier



#utilisation
  l utilisation est tres simples
  l utilisateur avec les coordonnées si dessous accede au menu principal
  puis il choisit entre 5 fonctions principales realisées par le programme
  ajouter un voyage/acceder a une description d'un voyage
  supprimer,modifier un voyage ou rechercher un voyage
